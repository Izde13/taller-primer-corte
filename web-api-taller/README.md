# Taller Web - Estructura Base

Estructura básica de página web responsiva.

## Estructura

```
web-api-taller/
├── index.html       # Página principal
├── css/
│   └── styles.css   # Estilos
├── js/
│   └── app.js       # JavaScript básico
└── assets/          # Imágenes (vacía por ahora)
```

## Características

- ✅ HTML semántico básico
- ✅ Navegación responsiva
- ✅ Grid de 3 columnas → 2 → 1
- ✅ Menú móvil funcional
- ✅ Skip link para accesibilidad

## Ver el proyecto

```bash
cd web-api-taller
start index.html
```

## Probar responsividad

1. Abrir DevTools (F12)
2. Modo responsive (Ctrl+Shift+M)
3. Probar diferentes tamaños

## Validar accesibilidad

- **WAVE**: https://wave.webaim.org
- **Lighthouse**: DevTools → Lighthouse
